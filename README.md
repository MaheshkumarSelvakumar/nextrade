# NexTrade — Capital Markets Dashboard

A full stack Capital Markets Transaction Dashboard built with:

- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Database:** MongoDB (coming soon)

## Features
- Real time trade monitoring
- Filter trades by status
- Search by symbol
- Sort by trade value
- Portfolio summary

## Running locally

### Backend
cd backend
pnpm install
node server.js

### Frontend
cd frontend
pnpm install
pnpm run dev
